# Random-Skill-Tree-Generator
Generates random skill tree with user inputs.

Written by Python3 and scikit-learn.

## What it can do
Not sure how to make a proper skill tree?

<img width="1127" height="602" alt="Pic 2025-12-14 065714" src="https://github.com/user-attachments/assets/67fee162-30c6-4a51-b9b5-a7c5bc3e38f7" />

Start with this random skill tree generator and boost your imagination!

All you need is:
- Number of first skills
- Number of final skills
- Maximum limit of tree height
- Maximum limit of tree width
- Extra rule: connection distance limit between skills

You can generate the skill trees until you get what you want, and download it!

## How to run

1. Set the virtual environment.
```
python -m venv venv
```

2. Run the app.py in the virtual environment.
```
python app.py
```
