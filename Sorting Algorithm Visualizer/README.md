# Sorting-Algorithm-Visualizer
Sorting algorithm visualizer with image and various algorithms.

Written by Python3.

## What it can do
You can upload the image, and visualize how the sorting algorithm sorts the randomized image!

![ȭ�� ��ȭ �� 2025-12-08 201336 (1)](https://github.com/user-attachments/assets/7714d949-7e55-461a-932b-d6c1fe23db31)

This program supports
- Bubble sort
- Insertion sort
- Selection sort
- Quick sort
- Merge sort
- Heap sort
- and Radix sort.

You can also add sound file to play every iteration!

There is also an secret image hidden somewhere...

## How to run

1. Set the virtual environment.
```
python -m venv venv
```

2. Run the app.py.
```
python3 app.py
```
