# PDF-Merger
PDF file merger with file order manipulation.

Written by Python3.

## What it can do
Getting tired of upload limitations and suspicious websites?

Merge your pdf files in your local computer!

## How to run

1. Set the virtual environment.
```
python -m venv venv
```

2. Enter the virtual environment and install the pypdf.
```
pip install pypdf
```

3. Run the app.py.
```
python app.py
```
